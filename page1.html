Some Concepts I have Learned So Far

  1. Making a Grid in CSS
    1. Grids are very useful in displaying the content you want to put on the webpage and are highly used.
    2. There are many different ways you can display your content on a website using grids.
    3. You can name the content area of a grid or the lines of a grid to specify where you want to place your items on the grid.
  2. Adding images to websites
    1. You can add images to a website by using the <img>. This tag will load the picture specified
    2. When adding an image to a webpage, you need to specify the path where the image is contained.
    3. You can add images by specifying the path of a url page or though a folder that the image is in attached to the project.
  3. Adding generic element tags
    1. Generic Elements can be useful when wanting to add scripts or style rules to your page
    2. Generic Elements are given semantic meanings with the class and id attribues which can be called back in the style and script pages.
    3. The <div> block indicates the division of content (generic blpck level).
    4. The <span> block indicates a word or phrase.
